# Course Learning Requirements (CLRs) with Embedded Knowlege and Skills (EKSs)

# CLR 1. Design and develop REST-compliant APIs

EKS 1.1 Identify the core characteristics of REST

EKS 1.2 Map API resources to application models

EKS 1.3 Map HTTP verbs to document CRUD operations

EKS 1.4 Create RESTful API services using a backend scripting language

# CLR 2. Understand the critical components of application security

EKS 2.1 Investigate options for client-application and user authentication

EKS 2.2 Create scripts that safely encrypt and store sensitive data

EKS 2.3 Create scripts that protect the transmission of sensitive data

EKS 2.4 Create scripts that sanitize incoming data before being stored

# CLR 3. Compare and contrast scalable persistent data storage technologies

EKS 3.1 Understand the strengths and weaknesses of relational databases

EKS 3.2 Understand the strengths and weaknesses of document databases

EKS 3.3 Learn about different document database management systems

# CLR 4. Write server-side scripts to manage and manipulate data

EKS 4.1 Install a document database system

EKS 4.2 Create scripts to store new data objects into the database

EKS 4.3 Create scripts to retrieve data objects from the database

EKS 4.4 Create scripts to modify existing data objects in the database

EKS 4.5 Create scripts to remove data objects from the database

# CLR 5. Integrate backend API service applications with document databases

EKS 5.1 Create a resource model schema for document databases

EKS 5.2 Recieve data objects from client applications and store them in the corresponding document collections

EKS 5.3 Respond to client application requests for data objects by returning correctly formatted JSON documents

EKS 5.4 Respond to client application requests for paginated datasets

# CLR 6. Compare and contrast various techniques for testing API services

EKS 6.1 Demonstrate manual API resources testing

EKS 6.2 Investigate various automated testing libraries

EKS 6.3 Create automated test scripts

EKS 6.4 Describe the goals and benefits of TDD (test driven development)

# CLR 7. Understand the options and benefits related to full-stack development

EKS 7.1 Compare and contrast various backend scripting languages

EKS 7.2 Investigate various backend application frameworks

EKS 7.3 Demonstrate how to integrate frameworks/packages/libraries into a backend application

# CLR 8. Gain an understanding of current best practices

EKS 8.1 Investigate various common design patterns

EKS 8.2 Learn to refactor code using modern language features
